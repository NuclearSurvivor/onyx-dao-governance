## Core Principle: Decentralization First

Every proposal in this package is designed around a single governing standard:

> **Accountability in decentralized systems is enforced by cryptographic proof, on-chain transparency, and structural independence — not by personal identity disclosure.**

Requiring leadership to be publicly identified (doxxed) is not a decentralization standard — it is a centralized institution standard that:
- Puts individuals in personal danger
- Creates a single point of legal/regulatory attack
- Has already been rejected by this community (Poll #67: "Should multisig signers be doxxed?" — **FAILED**)
- Contradicts how every major decentralized protocol operates (Bitcoin, Ethereum, MakerDAO, Compound, Uniswap — all pseudonymous contributors)

Accountability is achieved through:
- Public wallet addresses with immutable on-chain history
- Smart contract enforcement of multi-party authorization (no single actor controls anything)
- Independent treasury audits with public findings
- Conflict-of-interest structural prohibitions
- Community veto power and recall mechanisms
- Transparent on-chain voting records

## Phase 1 — Vote On These First

Four proposals. Immediate priority. The most urgent, most straightforward, and most impactful.

### Poll 1.1 — Codify 1,000,000 XCN Governance Threshold
The 1M threshold is already in place. This poll formally codifies it — preventing leadership from raising it without a community Supermajority vote.

### Poll 1.4 — Enforce the Justin Sun Lockup Vote
The community voted 100% FOR requiring a verified lockup smart contract in May 2025. No contract address has been published. This sets a 14-day binding deadline.

### Poll 1.5 — Independent Treasury Audit
16+ billion XCN distributed. Zero independent audits. Budget: up to 50M XCN. Public deliverable. This is not an accusation — it is basic fiduciary hygiene.

### Poll 1.6 — Minimum Quorum Standards
Community improvement polls with 97-100% approval fail at 27-53% quorum while insider spending proposals hit 149-319%. Tiered minimum quorum requirements close this loophole.
